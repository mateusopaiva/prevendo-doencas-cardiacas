## 📂 Descrição
Neste projeto, apliquei técnicas de machine learning, utilizando o algoritmo Random Forest, para prever o risco de doenças cardíacas em um grupo de pacientes.

A análise incluiu a coleta e tratamento de dados clínicos, permitindo a criação de um modelo preditivo capaz de identificar fatores de risco associados a doenças cardíacas. Este trabalho demonstra como a inteligência artificial pode ser aplicada à medicina para apoiar diagnósticos, prevenções e tratamentos mais precisos e eficazes, oferecendo soluções estratégicas para a saúde.

## 💻 Executando
```bash
cd prevendo-doencas-cardiacas
jupyter notebook prevendo_doencas_cardiacas_com_machine_learning.ipynb
```
